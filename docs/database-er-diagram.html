⚡ Arquitectura Dual-Database

Este proyecto usa dos bases de datos Supabase independientes. APEX IA Supabase almacena la infraestructura del SaaS (organizaciones, billing, roles). CLIENTE Supabase almacena los datos operacionales de cada tenant (conversaciones, contactos, deals). La conexión entre ambas se realiza via organizations_db.encrypted_database_url → resuelto dinámicamente por DatabaseProvider.getClientDrizzle(organizationId) con cache de 15 min.
